Local backup tools
Use this to keep a safe copy of your workouts, move data between iPhone and desktop, or reset local browser data.
  • Export on one device, then import the JSON file on another device if you want a manual backup or one-time transfer.
  • Local data is stored in your current browser only.
Cloud sync (Firebase Realtime Database)
Fill in your Firebase web app config + database URL. Once connected, this app can push local changes to the cloud and receive remote updates from your other devices.
  • Use the same Sync Key on iPhone and desktop to point both devices to the same dataset.
  • After connecting, remote updates from another device can be applied automatically while this page is open.
